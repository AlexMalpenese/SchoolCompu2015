HTML

HTML is what is used to make webpages and it stands for Hypertext Markup Language. HTML is important because without it your website will not show at all. HTML was created by Tim Berners-Lee. Most of this webpage is coded in HTML. There are many different commands used in HTML.

  • In HTML when you start a command the way to end the command is by adding a / infront of the command but inside the < and >.
  • <p> is used for adding paragraphs into your webpage. Like all the othe commands, </p> is used to end the paragraph.
  • The command <ul> is used to make an unordered list. An unordered list uses bullet points instead of numbers for each phrase added. </ul> is used to end the list.
  • <ol> is used to make an ordered list. An ordered list numbers each point added instead of making bullet points. </ol> is used to end the list.
  • <li> is used to add an item to a list. </li> is used to end the item.
  • <h1> is used is used to make a header. </h1> is how you would end the header.
  • <body> is the middle of your code. Images, words and all that stuff takes place in the body, </body> ends it.
  • To make an ID you would add id="Name of ID". You would add this inside of the beggining commad of what you wanted to change or add to. Example <p id="Name of ID"> Whatever you wrote about <p>. This would mean that if you added something to the ID you would change what you put inside the paragraph.
  • If you wanted to add a link that goes to the bottom of your page you would first need to add an ID to where you want the link to take you to within your webpage. Then you would add <a href="#Name of your ID"> what you want your link to say</a> This will take you to the ID you specified in the command. If you want the link to take you to another webpage you would just add the URL instead of the ID.
  • If you want to group multiple, non connecting, things then you need to make something called a class. to make a class you need to put class="name of class "inside of the command before the phrases you want to group.
  • If you want to add an image you have to use another command. <img src="url of image">
