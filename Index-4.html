Computer

We took apart a Dell computer in class. The computer that we took apart did not turn on in the beggining and it did not turn on afterward either. We still got to see most of the components of the computer.



This is what the computer looked when it is compleatly together before we touched anything. It looked like this when we put it back together as well



This is the motherboard of the computer. There was a lot of stuff that was on top of it and we had to clear it before we could actually take out the motherboard. The motherboard connects all of the other components in the computer together.



This is the CPU of the computer. CPU stands for central processing unit and it controlls the other parts of the computer.



This is the RAM of the computer. RAM stands for Random Access Memory. It stores data temporarily.



This is the hard drive of the computer. It is used to store and retrieve more data.



The different parts of the computer communicate with each other through many wires that connect to the motherboard.



This is a keyboard. The keyboard allows users to type letters, numbers, and symbols into the computer.



This is the mouse. The mouse allows users to be able to freely click on things and move things around. It makes it easier for the user to use the computer.



This is a monitor. THe monitor conncets to the computer and allows the user to see what they are doing within the computer. It is a major part of user interface.
