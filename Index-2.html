CSS

CSS is like HTML's support. CSS is used to style and move objects. Without CSS most websites would end up looking the same with the same font and same colors. Some of the commands require you to do some stuff using html first.

  • When you want to style the whole webpage as a whole you would have to put all your code within a body command. body{commands}
  • If you want to change the style of a certain phrase then first you would need to make an id for the phrase in html. Then instead of writing body, you should write #Whatever you named your id{commands}
  • If you want to change the style of multiple phrases then you need to add each phrase to a class in html. Then you would type .whatever you named the class{commands}
  • background-color:color; changes the color of the background. If you want to fine tune your color you would end using background-color: rgb( number,number,number) Each number controlls one of the primarey colors.
  • color:color; changes the color of the text. If you want to fine tune your color you would end using color: rgb( number,number,number) Each number controlls one of the primarey colors.
  • font-size:choose how big you want your text to be-represented in numbers-px; This changes the size of the text. To make the text bigger, make the number larger.
  • font-family:name of a font; changes the font of the text
  • width:numberpx(or %); changes the width of an image if it is under the id or class for images. If you use % instead of px then nomatter how big the screen gets, the image will be in poportion with the size of the screen. You can do the same thing but for height by switching out the word width with height
  • To make an image move you need to use multiple commands.
    • position:absolute; will make it so that you can move the image
    • float:left; makes it so that the image goes to the left of the page. If you want it to go to the right then you subsitute the left for a right. You can use this command without making the position absolute but that means you cant really move it around much.
    • margin:numberpx(or %); gives the picture some extra room around the edges. It will keep the image off the wall and stuff like that. If you want to have the margin on one side then you would add a -left,right,top,or bottom to the margin before the :.
    • top:numberpx; makes it so that the image is the number you put in away from the top of the page. you can subsitute the top for botton, left, or right to position your picture perfectly.
    • z-index:number; orders the images so you can make them overlap however you want. A z-index of 1 would be on the bottom.
